How it works

Guided setup, then one-click recording.

No account creation. No external meeting bot.

01

Run onboarding

Choose setup mode, grant permissions, and pick your first Whisper and AI models.

02

Record from app or overlay

Control recording from the main window or the floating overlay when the app is minimized.

03

Get transcript and notes

Whisper transcribes locally; AI summaries can run via Ollama or optional cloud providers.

Shipped Features

Built and available today.

These are implemented in the desktop app right now.

First-run onboarding

Guided setup with mode selection, permission checks, model download, and skip-AI option.

Local audio capture

Captures microphone and system audio together, with pause/resume and local WAV storage.

Whisper model manager

Download and switch `tiny`, `base`, `small`, and `medium` directly from onboarding or settings.

AI provider flexibility

Use local Ollama models or switch summaries to OpenAI/Anthropic with your own API keys.

Overlay-only workflow

Minimize or hide the main app and keep recording controls in a compact floating overlay.

Re-transcribe anytime

Re-run transcription + summarization from the stored audio using a different model or provider.
