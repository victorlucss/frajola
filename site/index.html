[ How it works ]

Three steps, nothing in the cloud

01

Run onboarding

Grant permissions, pick your Whisper model, and optionally set up Ollama for local AI summaries.

02

Record from app or overlay

Start recording in the main window. Minimize it and the floating overlay keeps you in control.

03

Get transcript and notes

Whisper transcribes locally. AI generates a summary with key points, decisions, and action items.

[ Features ]

Built and available today

Guided onboarding

First-run setup walks you through permissions, model downloads, and AI configuration. Choose full local, transcript-only, or cloud summaries.

Local audio capture

Captures microphone and system audio together as a single WAV file. Pause, resume, and stop from the app or the overlay.

Whisper model manager

Download and switch between tiny, base, small, and medium Whisper models directly from settings.

AI provider flexibility

Use local Ollama models for fully private summaries, or switch to OpenAI and Anthropic with your own API keys.

Floating overlay

Minimize the app and a compact floating pill stays on screen with recording controls, duration, and waveform.

Re-transcribe anytime

Re-run transcription and summarization from the stored audio using a different model or provider.

[ Processing modes ]

Local by default, cloud optional

Transcription is always local Whisper. Cloud usage only applies to AI summaries, and only when you enable it.
