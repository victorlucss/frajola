Changelog

Release history for Frajola.

0.1.1 March 5, 2026 Latest

Features

  • Onboarding flow with guided setup for permissions, Whisper models, and AI providers
  • Floating overlay pill for recording controls when app is minimized
  • Audio player for meeting playback
  • Meeting auto-detection (experimental)
  • Dark/light theme support

Improvements

  • Improved audio capture reliability on macOS
  • Better error handling for Whisper transcription
  • Settings page redesign with tabbed interface
  • Reduced memory usage during recording

Bug Fixes

  • Fixed overlay not appearing on some Windows configurations
  • Fixed transcript timestamps being off by a few seconds
  • Fixed crash when switching Whisper models during transcription
0.1.0 February 28, 2026

Initial Release

  • Local audio recording (microphone + system audio)
  • Whisper-based local transcription
  • AI-powered meeting summaries (Ollama, OpenAI, Anthropic)
  • Meeting library with search
  • Export to Markdown and plain text
  • macOS, Windows, and Linux support
